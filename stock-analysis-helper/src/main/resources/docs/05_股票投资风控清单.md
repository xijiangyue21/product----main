# 股票投资风控清单

> 说明：本文是股票投资风控学习资料，不推荐具体股票，也不构成买卖建议。

## 一、股票投资最大的风险是什么？

不是短期下跌，而是：

```text
你不知道自己为什么买，
也不知道什么时候该卖。
```

股票投资常见风险：

- 公司基本面恶化
- 行业周期下行
- 估值过高
- 财务造假
- 流动性差
- 情绪化交易
- 重仓单一股票
- 借钱投资

## 二、买入前必须回答的 7 个问题

## 1. 这家公司是做什么的？

- [ ] 主营业务是什么？
- [ ] 靠什么赚钱？
- [ ] 客户是谁？
- [ ] 产品有没有竞争力？

## 2. 它为什么值得买？

- [ ] 业绩增长
- [ ] 估值合理
- [ ] 行业前景好
- [ ] 分红稳定
- [ ] 资产质量好
- [ ] 只是别人推荐

如果答案只是“别人说会涨”，不要买。

## 3. 最大风险是什么？

- [ ] 业绩不达预期
- [ ] 行业政策变化
- [ ] 竞争加剧
- [ ] 负债过高
- [ ] 估值太贵
- [ ] 大股东减持
- [ ] 财务风险

## 4. 我准备买多少仓位？

建议新手不要单只股票过度集中。

参考：

```text
单只股票仓位不要让一次判断错误毁掉整个账户。
```

## 5. 如果下跌 20%，我怎么办？

提前写好规则：

- [ ] 基本面没变，继续持有
- [ ] 基本面变差，减仓
- [ ] 买入逻辑错误，止损
- [ ] 不知道怎么办，说明买入前没想清楚

## 6. 我什么时候卖出？

常见卖出理由：

- 买入逻辑被证伪
- 估值明显过高
- 发现更好的机会
- 需要调整资产配置
- 达到自己的风险上限
- 公司基本面恶化

不建议的卖出理由：

- 今天跌了很害怕
- 看到别人说不行
- 想追另一个热点
- 没有计划地频繁操作

## 7. 我是否用了短期要用的钱？

如果这笔钱 1 年内要用，不适合重仓股票。

## 三、股票仓位控制建议

## 新手阶段

目标：

```text
先学习，不追求赚快钱。
```

建议：

- [ ] 单只股票小仓位
- [ ] 不融资
- [ ] 不满仓
- [ ] 不追涨杀跌
- [ ] 每次买入写理由

## 进阶阶段

可以逐步学习：

- 财报分析
- 行业研究
- 估值方法
- ROE
- 毛利率
- 现金流
- 市盈率 PE
- 市净率 PB

## 六、风控红线

以下情况尽量不要做：

- [ ] 借钱炒股
- [ ] 满仓单只股票
- [ ] 听消息买入
- [ ] 不看财报就买
- [ ] 追连续大涨的热点
- [ ] 亏损后不断加仓摊平但没有逻辑
- [ ] 为了回本而加大赌注

## 七、股票投资一句话原则

```text
买股票前先想亏损，想清楚最坏情况，再决定能不能买。
```
