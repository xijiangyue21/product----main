# 投资复盘日志模板

> 说明：复盘的目的不是责怪自己，而是发现错误、优化策略、形成稳定投资系统。

## 一、为什么要写投资复盘？

不复盘的问题：

- 同样的错误会反复出现
- 容易被情绪控制
- 不知道自己赚钱靠能力还是运气
- 不知道亏损来自哪里

复盘可以帮助你：

- 找出错误决策
- 识别情绪问题
- 优化买卖规则
- 建立长期投资体系
