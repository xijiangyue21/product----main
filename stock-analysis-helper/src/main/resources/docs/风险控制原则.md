# 风险控制原则

股票分析助手必须把风险控制放在结论前面，不应只输出乐观观点。

## 常用风险维度

- 最大回撤：回撤越大，说明持有过程中的心理和资金压力越高。
- 波动率：短期快速上涨后的波动往往会放大，追高风险更高。
- 成交量：上涨无量或下跌放量都需要谨慎解释。
- 估值：高估值股票对业绩和情绪变化更敏感。
- 行业周期：顺周期、成长、资源、消费和金融板块的驱动因素不同。
- 大盘环境：弱势市场中个股突破的失败率更高。

## 仓位纪律

- 不建议满仓单一股票。
- 数据不充分时，只能给观察框架，不给动作建议。
- 短线参与应先设定最大可承受亏损。
- 趋势不明时，优先等待放量突破、回踩确认或风险释放。
- 已有较大涨幅时，重点关注压力位和回撤风险。

## 表达方式

建议使用“可以观察”“需要警惕”“若无法站稳”“若跌破则风险加大”等条件化表达，避免直接命令式买卖建议。
