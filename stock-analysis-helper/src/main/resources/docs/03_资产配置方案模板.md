# 资产配置方案模板

> 说明：资产配置是把资金分配到不同类别的资产中，用来平衡收益和风险。本文只提供通用模板，不构成个性化投资建议。

## 一、什么是资产配置？

资产配置就是决定你的钱分别放在哪里。

常见资产类别：

- 现金
- 货币基金
- 债券 / 债券基金
- 指数基金 / ETF
- 股票
- 黄金
- 其他资产

简单理解：

```text
不要把所有鸡蛋放在一个篮子里。
```

## 二、先把资金分成 4 个账户

## 1. 生活账户

用途：

- 日常开销
- 房租
- 交通
- 饮食
- 学费
- 水电

建议：

- [ ] 这部分钱不用于高风险投资

## 2. 应急账户

用途：

- 失业
- 生病
- 突发支出
- 家庭紧急情况

建议：

```text
至少准备 3～6 个月生活费。
```

适合放在：

- 银行活期
- 货币基金
- 高流动性低风险产品

## 3. 稳健账户

用途：

- 1～3 年内可能要用的钱
- 低风险增值
- 降低组合波动

可以学习：

- 短债基金
- 债券基金
- 固收类产品
- 低风险现金管理工具

## 4. 长期投资账户

用途：

- 5 年以上不用的钱
- 长期增值
- 养老、长期财富积累

可以学习：

- 宽基指数基金
- ETF
- 优质股票
- 股债组合

## 三、三种常见配置模板

## 1. 保守型模板

适合：

- 风险承受能力低
- 投资经验少
- 短期可能用钱
- 不喜欢大幅波动

示例：

| 资产类别 | 比例 |
|---|---:|
| 现金 / 货币基金 | 40% |
| 债券 / 债券基金 | 40% |
| 指数基金 / ETF | 15% |
| 学习型股票仓位 | 5% |

## 2. 稳健型模板

适合：

- 能接受一定波动
- 投资期限 3～5 年以上
- 想长期增值但不想太激进

示例：

| 资产类别 | 比例 |
|---|---:|
| 现金 / 货币基金 | 20% |
| 债券 / 债券基金 | 30% |
| 指数基金 / ETF | 40% |
| 股票 / 行业基金 | 10% |

## 3. 进取型模板

适合：

- 投资期限 5 年以上
- 能接受较大波动
- 有学习和复盘能力
- 不依赖这笔钱解决短期生活问题

示例：

| 资产类别 | 比例 |
|---|---:|
| 现金 / 货币基金 | 10% |
| 债券 / 债券基金 | 15% |
| 指数基金 / ETF | 55% |
| 股票 / 行业基金 | 20% |

## 五、再平衡规则

再平衡就是当比例偏离目标太多时，把组合调回原计划。

例如：

```text
目标：股票类资产 50%
实际：股票涨多后变成 65%
操作：适当卖出一部分，回到目标比例附近
```

我的再平衡规则：

- [ ] 每 3 个月检查一次
- [ ] 每 6 个月检查一次
- [ ] 当某类资产偏离目标比例超过 10% 时调整
- [ ] 不因为短期涨跌频繁调整

## 六、注意事项

- [ ] 不要频繁换策略
- [ ] 不要因为短期收益不好就否定长期计划
- [ ] 不要把短期资金放进高波动资产
- [ ] 不要只看收益率，也要看最大回撤
- [ ] 不要使用借贷资金投资

## 七、一句话总结

```text
资产配置的目的不是让你每次都赚最多，而是让你在可承受风险下长期活下来并积累收益。
```
