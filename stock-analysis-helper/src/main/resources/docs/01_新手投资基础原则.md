# 新手投资基础原则

> 说明：本文是通用投资学习资料，不构成个性化投资建议。投资前请结合自己的收入、负债、家庭责任、风险承受能力和投资期限进行判断。

## 一、投资前先完成 5 件事

- [ ] 留出 3～6 个月生活费作为应急资金
- [ ] 确认没有高利率负债，例如高息消费贷、信用卡长期欠款
- [ ] 明确投资目标：保值、增值、买房、养老、教育金等
- [ ] 明确投资期限：1 年内、3 年内、5 年以上
- [ ] 明确自己最多能接受多大亏损

## 二、新手最容易犯的错误

### 1. 一上来就重仓单只股票

新手常见错误是听到别人推荐某只股票，就一次性买入很多。
这会导致风险过于集中。

更稳妥的做法：

- [ ] 先学习基本概念
- [ ] 先小资金试错
- [ ] 不把全部资金压在一个股票、行业或主题上

### 2. 把短线投机当成投资

真正的投资更看重：

- 企业质量
- 长期现金流
- 行业空间
- 估值是否合理
- 自己是否能承受波动

短线频繁交易更容易受到情绪影响。

### 3. 看到上涨才追进去

很多亏损来自：

```text
上涨时贪心追高
下跌时恐慌割肉
反复高买低卖
```

所以投资前要先写好计划，而不是边涨边想。

## 三、普通人更适合的投资思路

## 1. 先求活下来，再求赚得多

投资第一目标不是暴富，而是避免重大亏损。

优先级：

```text
不亏大钱 > 稳定积累 > 提高收益
```

## 2. 分散投资

分散投资不是随便买很多个产品，而是把资金分散到不同资产：

- 现金
- 债券 / 货币基金
- 指数基金 / ETF
- 股票
- 黄金或其他资产

## 3. 长期投资

长期投资不是买了就不管，而是：

- 有计划地买入
- 定期检查组合
- 控制仓位
- 不因短期波动随意改变策略

## 四、新手投资顺序建议

## 第一步：学习基础概念

- [ ] 股票是什么
- [ ] 基金是什么
- [ ] ETF 是什么
- [ ] 债券是什么
- [ ] 指数是什么
- [ ] 分红是什么
- [ ] 市盈率 PE 是什么
- [ ] 市净率 PB 是什么

## 第二步：先做模拟盘或小资金

- [ ] 用小金额体验买入和卖出
- [ ] 记录每次买入理由
- [ ] 记录每次亏损原因
- [ ] 观察自己是否容易冲动

## 第三步：建立长期组合

可以考虑把资金分成几类：

```text
应急资金：不投资高风险资产
稳健资金：低风险产品
长期资金：指数基金 / ETF / 优质股票
学习资金：用于小额试错
```

## 五、投资前自查表

- [ ] 我是否知道这个产品靠什么赚钱？
- [ ] 我是否知道最坏情况下会亏多少？
- [ ] 我是否能接受 20% 以上的短期波动？
- [ ] 我是否有明确的买入理由？
- [ ] 我是否有明确的卖出条件？
- [ ] 我是否只是因为别人推荐才买？
- [ ] 我是否用了未来短期要用的钱投资？
- [ ] 我是否借钱投资？

## 六、核心原则总结

```text
1. 不懂不买
2. 不借钱投资
3. 不重仓单一资产
4. 不追热点梭哈
5. 不把短期波动当成长期结论
6. 先学习，再小资金试错，最后形成自己的体系
```
