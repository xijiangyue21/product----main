# 风险承受能力评估表

> 说明：本文用于帮助你判断自己更适合保守型、稳健型还是进取型投资方式，不构成个性化投资建议。

## 一、为什么要先评估风险？

投资不是只看收益，还要看你能不能承受亏损。

同样是亏损 20%：

- 对长期资金来说，可能还能等待恢复
- 对短期买房资金来说，可能会影响现实生活
- 对借来的钱来说，可能会造成严重压力

所以投资前先问自己：

```text
这笔钱亏掉一部分，会不会影响我的生活？
```

## 四、风险类型判断

## 保守型

特征：

- 不愿意承受明显亏损
- 投资期限较短
- 对波动很敏感
- 更看重本金安全

适合重点学习：

- 现金管理
- 货币基金
- 短债基金
- 低风险理财
- 少量指数基金体验

## 稳健型

特征：

- 能承受一定波动
- 投资期限一般在 3 年以上
- 希望在控制风险的前提下增值

适合重点学习：

- 指数基金
- ETF
- 债券基金
- 股债搭配
- 定投策略

## 进取型

特征：

- 能承受较大波动
- 投资期限较长
- 有一定学习能力和复盘能力
- 不会因短期下跌盲目操作

适合重点学习：

- 股票基本面分析
- 行业研究
- 指数增强
- 组合管理
- 风险控制
